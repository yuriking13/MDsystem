# Альтернативы для режима страниц

## Проблема
TipTap не поддерживает настоящую пагинацию - весь контент в одном потоке.

## Решения

### 1. Pagedjs (Рекомендуется)
- **Библиотека**: https://pagedjs.org/
- **Установка**: `pnpm add pagedjs`
- **Описание**: Превращает HTML в paged media для печати
- **Плюсы**: Автоматическая пагинация, CSS Paged Media
- **Минусы**: Только для чтения/печати, не для редактирования

### 2. Lexical (от Facebook/Meta)
- **Библиотека**: https://lexical.dev/
- **Установка**: `pnpm add lexical @lexical/react`
- **Описание**: Современный фреймворк редактора
- **Плюсы**: Лучшая производительность, гибкая архитектура
- **Минусы**: Нужна миграция с TipTap

### 3. CKEditor 5
- **Библиотека**: https://ckeditor.com/
- **Установка**: `pnpm add @ckeditor/ckeditor5-react @ckeditor/ckeditor5-build-classic`
- **Описание**: Мощный WYSIWYG редактор
- **Плюсы**: Есть плагины для страниц
- **Минусы**: Тяжелее, коммерческая лицензия для некоторых features

### 4. Quill с пагинацией
- **Библиотека**: https://quilljs.com/
- **Установка**: `pnpm add react-quill`
- **Описание**: Легкий редактор
- **Плюсы**: Простой, легкий
- **Минусы**: Нужен кастомный модуль для страниц

### 5. ProseMirror напрямую (как TipTap, но с контролем)
- **Библиотека**: https://prosemirror.net/
- **Описание**: То, на чем построен TipTap
- **Плюсы**: Полный контроль, можно сделать пагинацию
- **Минусы**: Много кода, сложная настройка

### 6. Slate.js
- **Библиотека**: https://www.slatejs.org/
- **Описание**: Фреймворк для кастомных редакторов
- **Плюсы**: Полный контроль
- **Минусы**: Нужно писать всю логику самому

## Быстрое решение (без замены редактора)

### Вариант A: Режим "Print Preview"
Использовать TipTap в обычном режиме, но добавить кнопку "Предпросмотр печати" которая:
1. Рендерит контент в iframe с CSS @page
2. Показывает разбивку на страницы
3. Можно печатать/экспортировать

### Вариант B: Визуальные линейки страниц
- Оставить текущий редактор как есть
- Добавить визуальные маркеры каждые 1123px (высота A4)
- Предупреждать о переносе элементов

### Вариант C: Paged.js для preview
- Редактирование в TipTap (scroll mode)
- Кнопка "Pages Preview" рендерит через Paged.js
- Экспорт в PDF через Paged.js

## Рекомендация
Для вашего случая (научные работы с графиками):
**Использовать TipTap + Paged.js**

1. Редактирование в scroll mode (убрать pages mode)
2. Добавить кнопку "Предпросмотр" 
3. Preview рендерится через Paged.js с правильной пагинацией
4. Экспорт PDF через Paged.js
