# Варианты лэндинга Scientiaiter

Три концептуальных варианта дизайна лэндинга, каждый из которых отражает уникальный подход к представлению идеи слияния классической науки и ИИ.

## 🎨 Варианты дизайна

### 1. Современный стиль (`/modern/`)

- **Концепция**: Будущее науки через технологии
- **Визуал**: Минимализм, чистые линии, градиенты
- **Цвета**: Синяя палитра (#3B82F6, #2563EB)
- **Типографика**: Inter, современные шрифты
- **Элементы**: Плавающий код, интерактивные карточки
- **Настроение**: Инновации, прогресс, эффективность

### 2. Советский стиль (`/soviet/`)

- **Концепция**: Системный подход к научной автоматизации
- **Визуал**: Конструктивизм, геометрия, промышленная эстетика
- **Цвета**: Красно-золотая схема (#C41E3A, #FFD700)
- **Типографика**: Arial Black, жирные шрифты
- **Элементы**: Производственная анимация, прогресс-бары
- **Настроение**: Планомерность, мощь, систематичность

### 3. Греческий стиль (`/greek/`)

- **Концепция**: Связь с философскими корнями науки
- **Визуал**: Классическая античность, академизм
- **Цвета**: Мраморно-золотая палитра (#F8F6F0, #D4AF37)
- **Типографика**: Cinzel, Crimson Text (классические засечки)
- **Элементы**: Древние свитки, греческие символы
- **Настроение**: Мудрость, традиции, философия

## 🎯 Общая концепция

Все варианты объединяет:

- **Основная идея**: Слияние классической научной методологии с возможностями ИИ
- **Специализация**: Фокус на медицинских исследованиях
- **Функционал**: Поиск литературы, анализ, написание, автоматизация
- **ЦА**: Исследователи, академические работники, медицинские специалисты

## 🚀 Просмотр

1. Откройте `index.html` для выбора варианта
2. Каждый вариант — полноценный статический лэндинг
3. Все ссылки ведут на существующие роуты приложения

## 📱 Адаптивность

Все варианты адаптированы под:

- Desktop (1200px+)
- Tablet (768px-1200px)
- Mobile (до 768px)

## 🔧 Технические детали

- Чистый HTML/CSS/JS, без зависимостей
- CSS Grid/Flexbox для layout
- Плавная анимация и transitions
- Семантическая разметка
- Оптимизированы под производительность

## 🎨 Использование кода и пиксель-арта

Каждый вариант включает визуализацию кода:

- **Современный**: Плавающие блоки кода с AI/ML терминологией
- **Советский**: Промышленные "модули" в стиле производственной линии
- **Греческий**: Древние свитки с псевдо-греческим кодом
